# 🛠️ Requisitos Funcionais (RFs)

Este documento contém todos os requisitos funcionais aplicados ao projeto LES E-commerce 2026. Toda nova funcionalidade deve ser registrada aqui para rastreabilidade.

| ID       | Requisito                         | Descrição                                                                                      | Status          |
| :------- | :-------------------------------- | :--------------------------------------------------------------------------------------------- | :-------------- |
| **[RF]** | **Múltiplos Endereços**           | Suporte nativo para múltiplos endereços (residencial e entrega) com cálculo de frete dinâmico. | ✅ Implementado |
| **[RF]** | **Carrinho Resiliente**           | Itens persistidos via `localStorage`, garantindo a recuperação da sacola em qualquer sessão.   | ✅ Implementado |
| **[RF]** | **Sincronização entre Abas**      | Redux + Storage Events para atualização em tempo real do estado global.                        | ✅ Implementado |
| **[RF]** | **Análise de Vendas (Dashboard)** | Visualização gráfica de métricas de receita e volume por período para administradores.         | ✅ Implementado |
| **[RF]** | **Gestão de Estoque**             | Controle administrativo de entrada e ajuste de quantidades de livros.                          | ⏳ Pendente UI  |
| **[RF]** | **Troca e Devolução**             | Funcionalidade de solicitação de troca integrada ao histórico de pedidos.                      | ⏳ Pendente UI  |
