# 🏗️ Requisitos Não-Funcionais (RNFs)

Este documento contém todos os requisitos não-funcionais aplicados ao projeto LES E-commerce 2026.

| ID        | Requisito                   | Descrição                                                                  |
| :-------- | :-------------------------- | :------------------------------------------------------------------------- |
| **[RNF]** | **Arquitetura React 19**    | Uso das funcionalidades modernas do React para máxima performance.         |
| **[RNF]** | **Cypress para Testes E2E** | Cobertura de 100% dos fluxos críticos com testes automatizados funcionais. |
| **[RNF]** | **Tipagem TypeScript**      | Rigorosa tipagem no domínio para evitar bugs em tempo de execução.         |
| **[RNF]** | **Zero CSS Inline**         | Toda estilização deve seguir o padrão de CSS Modules.                      |
